## 🚀 安装方式

### Chrome 应用商店
1. 访问 [Chrome 应用商店](https://chrome.google.com/webstore/detail/websocket-devtools/your-extension-id)
2. 点击 **"添加至 Chrome"** 并确认安装
3. 打开开发者工具 (F12) → **"WebSocket DevTools"** 选项卡

### 手动安装  
1. 从 [GitHub Releases](https://github.com/law-chain-hot/websocket-devtools/releases) 下载
2. 打开 `chrome://extensions/` → 启用 **开发者模式**
3. 点击 **"加载已解压的扩展程序"** 并选择文件夹

## 📖 快速开始

1. **打开开发者工具** (F12) → 找到 **"WebSocket DevTools"** 选项卡
2. **开始监控** - 自动检测 WebSocket 连接
3. **查看消息** - 点击连接查看消息历史  
4. **模拟消息** - 使用模拟选项卡和 JSON 编辑器
5. **保存收藏** - 收藏常用消息以便快速访问

## 🔒 隐私与安全

- ✅ **不收集数据** - 所有数据都保存在您的浏览器中
- ✅ **仅本地存储** - 不使用外部服务器或跟踪
- ✅ **开源透明** - 代码完全透明可审计
- ✅ **最小权限** - 仅申请功能必需的权限

## 🛠 系统要求

- **Chrome 88+** 或基于 Chromium 的浏览器（Edge、Brave 等）
- **WebSocket API** - 兼容所有 WebSocket 实现
- 支持 **Socket.IO**、**ws 库** 和自定义 WebSocket 解决方案
